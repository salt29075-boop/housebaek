이메일 알림 설정 (EmailJS)

1
EmailJS 가입 — emailjs.com에서 무료 가입 (월 200건 무료)
2
Email Service 연결 — 좌측 메뉴 Email Services → Add New Service → Gmail(또는 네이버 등) 선택 후 연결. 생성된 Service ID 복사
3
Email Template 생성 — Email Templates → Create New Template. 본문에 아래 변수 사용 가능:
{{name}} 이름   {{phone}} 연락처   {{type}} 시공유형   {{budget}} 예산   {{message}} 문의내용   {{date}} 접수일시. 저장 후 Template ID 복사
4
Public Key 복사 — 우측 상단 계정 → Account → Public Key 복사
5
위 입력란에 붙여넣기 → 설정 저장 클릭. 이후 상담 신청 접수 시 지정 이메일로 자동 발송됩니다.
✅ 저장되었습니다. 랜딩 페이지를 새로고침하면 반영됩니다.
